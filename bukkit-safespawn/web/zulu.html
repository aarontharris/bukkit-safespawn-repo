Quick: How to create a teleporter

Teleporters are created by magically enhancing ( /cast charge or /cast lock ) two wall signs with the command teles on the first line and any name you like on the second line. Two teleporters with the same name will be linked. The name is unique to each player so don't worry if your teleporter name is the same as someone else's.

    Requirements:
  1. The teleporter sign must be raised at least one unit off the ground or you will be partially teleported into the ground.
  2. All wall signs must be paired with another of the same name, or else how will it know where to send you?
  3. Both teleporter signs must be magicaly enhanced.
  4. Both teleporter signs must be placed on a block that does not observe gravity -- no sand or gravel, etc.
  5. Requires xp level 1 (will be consumed)
    Steps (refer to images above):
  1. Place two blocks vertically.
  2. Place a sign on the top block.
  3. Sign Text Line 1: "teles"
  4. Sign Text Line 2: Enter a teleporter name
  5. Sign Text Line 3: (optional)
  6. Sign Text Line 4: (optional)
  7. Point the cursor at the sign (make sure nothing else is in the way)
  8. Now enhance the block by typing: /cast charge
  9. Repeat these steps for the second teleporter, be sure to use the same name.
  10. Once both teleporters are properly created, you may teleport between them by clicking the sign

Quick: How to lock a teleporter

Locking a teleporter allows you to control who may use the teleporter. You may lock a teleporter so that only you and those you grant access to may use it. These instructions assume you already have a working teleporter set up and you are the owner. If you are not the owner, you will not be able to lock the teleporter.

    Requirements:
  1. A working teleporter system.
  2. You are the owner.
  3. Requires xp level 1 (will be consumed).
    Steps to lock only allowing yourself access (refer to images above):
  1. Point the cursor at the sign (make sure nothing else is in the way)
  2. Now enhance the sign by typing: /cast lock (see image 1 above)
  3. Repeat these steps for the second teleporter, be sure to use the same name.
  4. Both teleporters are now locked so that only you may use them by default, for more see locking access.
  5. Test your lock by pointing at the sign and typing: /cast access (see image 2 above)

Quick: Locking Access - How to grant or revoke access to a lock

Once you have successfully locked a teleporter or a chest, etc, you may now grant or revoke access for others. You may only alter access if you are the owner. If you are not the owner, you will not be able to alter the lock.

    Requirements:
  1. A working lock, test your lock by pointing at the sign and typing: /cast access (see image 1 below)
  2. You are the owner.
  3. No xp required to alter access.
    Steps to grant or revoke access for others (refer to images above):
  1. Point the cursor at the sign (make sure nothing else is in the way)
  2. Now grant access by typing: /cast grant playername (see image 2 above)
  3. /cast access again to confirm your change.
  4. Repeat these steps for the second teleporter, as both teleporters must have the same access.
  5. You have not granted access to "playername"
  6. You may also revoke access the same way with /cast revoke playername (see image 3 above)
